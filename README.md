![logo](IMG_20191012_135437.jpg)
#### Installing

1. git clone https://github.com/CharlieTheHack1/Cute-Termux
2. chmod +x *
2. ./setup.sh

#### Updating

1. use command **updatedw** to update the script.

#### Uninstalling
1. Run the script again to uninstall styling.
